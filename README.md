# SAIRAM-AI-CHATBOT-APPLICATION
AI CHATBOT APPLICATION:

STEP 1: Project Overview
This is an AI-powered chatbot application built using Python Flask and Google Gemini API.
Users interact with the chatbot through a web interface, and the chatbot responds intelligently using Geminiâ€™s generative AI model.

STEP 2: Features
The application provides a simple chat UI, a Flask backend API, secure API key handling using environment variables, and is ready for local use or deployment on platforms like Render.

STEP 3: Technologies Used
The backend is built with Python and Flask.
Google Gemini (gemini-2.5-flash) is used for AI responses.
Frontend uses HTML, CSS, and JavaScript.
Environment variables are managed using python-dotenv.

STEP 4: Project Folder Structure
The main Python file runs the Flask server.
Templates folder contains index.html for UI rendering.
Static folder holds CSS or JS files if needed.
.env file stores the API key securely.
requirements.txt contains dependencies.

STEP 5: Setup Instructions
First, clone the project from GitHub to your system.
Next, install all required Python packages using requirements.txt.
Create a .env file and add your Google API key.
Run the Flask application using Python.
Open the browser and access the local server URL to use the chatbot.

STEP 6: Deployment
For deployment on Render, add the API key in environment variables and set the start command to run the Flask app.
LIVE DEPLOYED LINK:- https://sairam-ai-chatbot-application-5.onrender.com/

STEP 7: Security
Never hardcode API keys in the source code.
Always use environment variables for sensitive data.
